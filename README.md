# Volatility Forecasting System

A production-grade MLOps pipeline for forecasting financial market volatility.

## Stack
- Data: yfinance + SQLite
- Modeling: scikit-learn, XGBoost, MLflow
- Serving: FastAPI
- Drift Detection: Evidently AI
- Orchestration: Prefect
- Dashboard: Streamlit
